# 🏍️ 와구와구 배달 기사 서버

### 👤 담당자 : 조진호

## <br>📃 핵심 기능
### 배달 기사, 내역, 요청 건 관리 / 배달 기사 실시간 위치 관리 및 거리 계산

- 배달 기사 정보 관리
- 배달 상세 내역 가져오기
- 배달 기사 자격 검증 후 배달 요청 건 할당
- 배달 기사 실시간 위치 저장 및 가져오기
- 두 지점 위,경도로 사이의 거리 계산
